Who We Are

ListingHub is a private internal application used to manage Etsy seller operations for a single shop.

Data We Access

  • Etsy shop profile and listing metadata
  • Order and transaction data required for operations
  • Inventory and fulfillment status for internal reporting

How We Use Data

  • To operate internal dashboards and reports
  • To improve listing quality and operational workflows
  • To monitor inventory and order processing

Data Sharing

We do not sell Etsy data. We do not provide public access to private shop data. Data is shared only with internal team members who need it for operations.

Security

  • OAuth tokens are stored securely
  • Access is restricted to authorized internal users
  • Secrets are not hardcoded in client-side code

Retention and Deletion

Operational data is retained only as needed for analytics and compliance. If API access is revoked, related tokens are deleted or invalidated.
